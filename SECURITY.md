# Security Policy

License: MIT (see LICENSE).

## Supported versions
Only the latest release line is actively supported with security fixes.

## Reporting a vulnerability
Please report vulnerabilities privately by email:
- 2000.aman.sinha@gmail.com

Please include:
- affected version
- reproducible steps or proof of concept
- impact assessment

Do not open public issues for unpatched vulnerabilities.

## Response targets
- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Fix and coordinated disclosure: timeline depends on severity and complexity
